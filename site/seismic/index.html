Test bed 2 · Southern California · two strike-slip faults

Does the quantizer
preserve the hazard?

Spectral-acceleration maps are simulated with the Abrahamson–Silva (1997) ground-motion model and Jayaram–Baker (2009) spatial correlation, then quantized with FQ-IDCVT. The test: do the weighted quanta reproduce marginal exceedance probabilities and spatial autocorrelation against a Monte-Carlo reference?

About these numbers. The figures below come from this study's recalculation notebook run in its fast reproducible configuration (224- and 504-site grids, N = 50, 2,000 MC maps) so the whole pipeline executes end-to-end in about a minute. The paper-scale run (1,116 / 2,484 sites, N = 200, 30,000 MC maps) reaches < 0.01 max exceedance error and ±0.02 autocorrelation error. The point demonstrated here is method transfer and qualitative fidelity, not the publication accuracy figure.

Result · the N-question

Why N = 50 already looks like N = 200

A natural question: if more quanta is better, why does N = 50 reproduce the hazard nearly as well as N = 200? Because the spatial structure is governed by the smooth median field, which cluster means capture even at N = 50, while marginal exceedance is already accurate there. Errors barely move; variance capture creeps up only slowly.

Bottom line. FQ-IDCVT reproduces both the marginal exceedance and the second-order spatial structure of the intensity field in 2-D — and it does so with any of the benchmarked clustering engines. The seismic application confirms what test bed 1 measured: you can pick the fast engine without paying in accuracy.
